Sizes of Created Encrypted file containers

Foreword: Created a file container and then tried to copy the files in and got informed the files couldn’t fit even though I’d left quite a bit of headroom. Then noticed a lot of space from what I selected was missing. This is the second time this happened to me and decided to write it down for next time.

Procedure

Objective create multiple containers all set to the same size to compare the container sizes produced

  1. Create an encrypted file container
  2. Standard VeraCrypt volume
  3. Volume Size
    1. 10 MiB
    2. 100 MiB
    3. 1000 MiB
  4. Select Format 2. Linux Ext4
    1. FAT
    2. NTFS
