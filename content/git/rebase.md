# Rebase onto develop

```sh
git rebase develop
```

If you'd already pushed the branch before you'll need to do a force push

```sh
git push -f
```

# Rebase interactive

Source: <https://git-scm.com/docs/git-rebase>

```sh
git rebase -i
```
